# Stacks and Queues

## Stack (LIFO)
-   **Last In First Out**.
-   Operations: `push`, `pop`, `peek`.
-   **Usage**: Function calls, undo mechanisms, expression parsing.
-   **Complexity**: O(1) for push/pop.

## Queue (FIFO)
-   **First In First Out**.
-   Operations: `enqueue`, `dequeue`, `peek`.
-   **Usage**: Task scheduling, BFS, printers.
-   **Complexity**: O(1) for enqueue/dequeue.

## Example Usage
```bash
python3 stack.py
python3 queue.py
```
